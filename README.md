# 📊 Teen Mental Health Analysis (Excel Project)

## 📌 Project Overview

This project analyzes how social media usage affects teen mental health using Excel.

## 🎯 Objectives

* Understand relationship between social media and stress
* Analyze sleep impact on mental health
* Compare platform usage (Instagram, TikTok)
* Identify patterns in depression among teens

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers (for dashboard)

## 📊 Key Insights

* Higher social media usage is linked with higher stress
* Lower sleep hours may increase depression risk
* Platform usage shows different mental health patterns

## 📁 Dataset

Teen Mental Health Dataset (CSV)

## 🚀 Conclusion

This project shows how data analysis in Excel can help understand real-world problems like mental health.
