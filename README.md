# 📊 Teen Mental Health Analysis (Excel Project)

## 📌 Project Overview

This project analyzes how social media usage affects teen mental health using Excel.

## 🎯 Objectives

* Understand relationship between social media and stress
* Analyze sleep impact on mental health
* Compare platform usage (Instagram, TikTok)
* Identify patterns in depression among teens

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers (for dashboard)

## 📊 Key Insights

* Higher social media usage is linked with higher stress
* Lower sleep hours may increase depression risk
* Platform usage shows different mental health patterns

## 📁 Dataset

Teen Mental Health Dataset (CSV)

## 🚀 Conclusion

This project shows how data analysis in Excel can help understand real-world problems like mental health.

🔷 PROJECT OVERVIEW
This project analyzes the impact of social media usage on:
Mental health (anxiety, stress, depression)
Sleep patterns
Addiction level
Academic performance
Focus: Age & Gender comparison
Goal: Identify patterns + give insights using interactive dashboard

🔷 DATASET
Source: Survey-based dataset (Kaggle / similar)
Records: ~1000 (approx)
Features include:
Age
Gender
Anxiety level
Addiction level
Sleep hours
Daily social media usage
Academic performance
Type: Structured data

🔷 DATA CLEANING & TRANSFORMATION
Removed missing values & duplicates
Standardized categorical values (Male/Female, platform names)
Converted data types (text → numeric where needed)
Created new columns:
Age groups
Average metrics (sleep, usage, etc.)
Cleaned inconsistent values for accurate analysis

🔷 SLICERS (Filters)

Used for interactivity:

Age
Gender
Anxiety Level
Addiction Level
Academic Performance
Daily Social Media Usage

Sleep Hours

Different charts used:

Bar Chart → Comparison (Male vs Female)
